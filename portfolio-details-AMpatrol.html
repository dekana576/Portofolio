Project information

  • Name: AM Patrol
  • Client: PT. Astra International Tbk - Honda
  • Project date: 06 August, 2025
  • Project URL: https://ampatrol.pgsasmodps.com/

AM Patrol is a digital platform designed to improve the efficiency and accuracy of security patrol operations. The system allows security personnel to scan QR code checkpoints, record inspection results, upload photos, and submit reports in real time. Each report includes automatic timestamps and location data for reliable verification. Administrators can review, approve, and monitor patrol activities through a centralized dashboard. With its structured workflow and user-friendly interface, AM Patrol ensures transparent, well-documented, and efficient security oversight across all locations.
